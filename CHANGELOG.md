# Changelog

All notable changes to the **Bauhaus** Obsidian theme will be documented in this file.

## [1.0.0] - 2026-02-27

### Added
- Initial release of the Bauhaus theme.
- Modern, minimalistic design system inspired by the Bauhaus movement.
- Light and Dark modes with curated tonal surfaces.
- Support for geometric typography (Jost).
- Custom styling for tags, buttons, and callouts following Material 3 principles.
- Clean, borderless workspace layout.
